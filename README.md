## Installation

Via environment variable:

    export ANSIBLE_ROLES_PATH=~/github/ansible-roles

Via [GNU Stow](https://www.gnu.org/software/stow/):

    mkdir -p ~/.ansible/roles
    stow --ignore='README.md' -t ~/.ansible/roles .
